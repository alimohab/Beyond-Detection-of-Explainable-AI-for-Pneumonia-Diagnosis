# Beyond-Detection-of-Explainable-AI-for-Pneumonia-Diagnosis
Explainable AI system for pneumonia detection using chest X-ray images. The project evaluates multiple deep learning models (CNNs, EfficientNet, MobileNet, ViT) and integrates the XAI-ICP framework to provide interpretable predictions through heatmaps and feature attribution, enhancing transparency and clinical trust.
